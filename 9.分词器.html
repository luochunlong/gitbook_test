9. 分词器

分词器

  • Analyzer（分词器）的作用是把一段文本中的词按一定规则进行切分。对应的是Analyzer类，这是一个抽象类，切分词的具体规则是由子类实现的，所以对于不同的语言，要用不同的分词器(也就是说不同的分词器分词的规则是不同的)。
  • 在创建索引时会用到分词器，在搜索时也会用到分词器，这两个地方要使用同一个分词器，否则可能会搜索不出结果。

  • 常用分词器

    • StandardAnalyzer（单字分词器，Lucene框架提供）

    • SmartChineseAnalyzer（智能中文分词器，Lucene框架提供）

    • IKAnalyzer（IK分词器，国内研发最流行的中文分词器）

  • 英文分词与中文分词的区别

    英文分词
    • 切分关键词.(用空格)
      • 例如：welcome to china 分词后的效果 welcome、 to 、 china 直接拆分成三个单词
    • 去除停用词
      • 有些词在文本中出现的频率非常高，但对文本所携带的信息基本不产生影响，例如英文的“a、an、the、of”，或中文的“的、了、着”，以及各种标点符号等，这样的词称为停用词（stop word）。 建立索引时停用词没有意义，因此不需要对停用词建立索引，检索信息时停用词也没有作用。

中文分词

  • 中文分词比较复杂，英文分词可按空格进行分词，但中文分词则很难，中文分词中每个汉字在不同的地方可以组成不同的词语。

  • 中文分词算法有： 单字分词（通过使用StandardAnalyzer分词器） 词库分词（通过使用SmartChineseAnalyzer或者IKAnalyzer），中文环境推荐使用词库分词方式，而IKAnalyzer分词器是目前国内中文环境下最受欢迎的中文分词器。

  • 不同分词器分词结果演示

    示例代码：

    ``` import org.apache.lucene.analysis.Analyzer; import org.apache.lucene.analysis.TokenStream; import org.apache.lucene.analysis.cn.smart.SmartChineseAnalyzer; import org.apache.lucene.analysis.standard.StandardAnalyzer; import org.apache.lucene.analysis.tokenattributes.TermToBytesRefAttribute; import org.apache.lucene.util.BytesRef; import org.wltea.analyzer.lucene.IKAnalyzer;

    public class IndexManager {

  public static void main(String[] args) {

      try {
          //定义需要分词的内容
          String str = "长沙有哪些好玩的旅游景点";

          // 使用单字分词
          Analyzer analyzer = new StandardAnalyzer();

          // 使用lucene提供的智能中文分词
          //Analyzer analyzer = new SmartChineseAnalyzer();

          // 使用国内研发的IK分词器
          //Analyzer analyzer = new IKAnalyzer();        

          // 对指定的字符串进行分词
          TokenStream ts = analyzer.tokenStream(null, str);

          //先要对tokenStream执行reset
          ts.reset(); 
          // 采用循环，不断地获取下一个token
          while(ts.incrementToken()) {

// // 获取其中token信息 TermToBytesRefAttribute attr = ts.getAttribute(TermToBytesRefAttribute.class); final BytesRef bytes = attr.getBytesRef(); //打印分词结果 System.out.println(bytes.utf8ToString()); } ts.close(); 

      } catch (Exception e) {
          // TODO: handle exception
          e.printStackTrace();
      }

  }

}

```

  • 使用StandardAnalyzer单词分词器后分词结果如下，也就是将每一个单个单个拆分开
    • 长 沙 有 哪 些 好 玩 的 旅 游 景 点

  • 使用SmartChineseAnalyzer智能中文分词器后分词结果如下，也就是按照SmartChineseAnalyzer词库中的单词进行匹配然后进行分词；

    • 长沙 有 哪些 好 玩 的 旅游 景点

  • 使用IKAnalyzer中文分词器后分词结果如下，也就是按照IKAnalyzer词库中的单词进行匹配然后进行分词，很明显IKAnalyzer分词风格会更加符合中国人的习惯，哪些 的 这种没有价值的信息都过滤掉了

    • 长沙 好玩 旅游景点 旅游 景点

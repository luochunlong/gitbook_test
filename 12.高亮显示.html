12. 高亮显示

高亮显示

  • 何为高亮显示呢，比如我们进入京东商城首页：在搜索栏中输入关键字 华为移动手机，进行查询的时候，显示出如下效果：

​ 查询到的结果中出现 华为 移动 手机 关键词的都变成了红色效果，这种效果的处理我们称之为高亮显示；

  • 高亮显示通常会把内容截取一段文本，用于生成摘要，并突出（高亮）显示关键词。 所谓高亮：其实就是通过为关键词添加前缀、后缀来实现。比如增加既可以将关键词加粗显示。
  • 高亮显示需要highlighter\lucene-highlighter-x.x.x.jar和memory\lucene-memory-x.x.x.jar两个目录下的jar包,在进行全文检索的时候调用高亮相关API即可实现高亮效果；

  • 高亮显示核心API

    • Formatter- 它是负责指定进行高亮的格式。比如SimpleHTMLFormatter(String preTag, String postTag)
    • Scorer－它用于包装查询，它里面包含了关键词信息。
    • Highlighter－进行高亮的工具类。 Highlighter(Formatter,Scorer) 调用Highlighter的getBestFragment即可获取高亮后的文本片段。
    • Fragmenter：用于截取高亮文本的其中部分。

  • 高亮显示的编程步骤

    1. 先要创建Formatter对象，它用于指定高亮的格式。
    2. 创建Scorer对象，它封装了Query对象。Scorer对象里包装的是关键词信息。
    3. 以Formatter、Scorer对象创建Highlighter工具。
    4. 调用Highlighter对象的getBestFragment来返回高亮文本。
    5. 可选的截断：调用Highlighter的setTextFragmenter()方法控制高亮文本的截断。
    6. 调用highlighter.getBestFragment()方法进行高亮数据的获取

  • 示例代码：

    ``` import java.nio.file.Paths; import org.apache.lucene.analysis.Analyzer; import org.apache.lucene.analysis.cn.smart.SmartChineseAnalyzer; import org.apache.lucene.document.Document; import org.apache.lucene.index.DirectoryReader; import org.apache.lucene.index.IndexReader; import org.apache.lucene.queryparser.classic.MultiFieldQueryParser; import org.apache.lucene.queryparser.classic.QueryParser; import org.apache.lucene.search.IndexSearcher; import org.apache.lucene.search.Query; import org.apache.lucene.search.ScoreDoc; import org.apache.lucene.search.TopDocs; import org.apache.lucene.search.highlight.Formatter; import org.apache.lucene.search.highlight.Fragmenter; import org.apache.lucene.search.highlight.Highlighter; import org.apache.lucene.search.highlight.QueryScorer; import org.apache.lucene.search.highlight.Scorer; import org.apache.lucene.search.highlight.SimpleFragmenter; import org.apache.lucene.search.highlight.SimpleHTMLFormatter; import org.apache.lucene.store.Directory; import org.apache.lucene.store.FSDirectory;

    public class IndexManager {

  public static void main(String[] args) {
      try {
          //指定索引库的目录
          Directory directory = FSDirectory.open(Paths.get("I:\\lucene_db\\tb_article"));

          //DirectoryReader的open方法指定需要读取的索引库信息，并返回相应的实例
          IndexReader indexReader = DirectoryReader.open(directory);

          //创建IndexSearcher对象，通过IndexSearcher对象进行全文检索实现
          IndexSearcher  indexSearcher = new IndexSearcher(indexReader);

          //创建智能分词器,对用户输入的信息进行分词   注意：索引的CRUD必须使用相同的分词器   
          Analyzer analyzer = new SmartChineseAnalyzer();

          //创建QueryParse实例   通过QueryParse可以指定需要查询的字段以及分词器信息
          QueryParser queryParser = new MultiFieldQueryParser(new String[]{"title","content"}, analyzer);

          //通过QueryParser得到query实例,并指定的检索的关键词
          Query query = queryParser.parse("长沙旅游指南");


          /**********高亮相关代码begin**********/
          //1、创建格式器，通过格式器指定怎样对关键字进行处理    
          Formatter formatter = new SimpleHTMLFormatter("<font color='red'>", "</font>");

          //通过Scorer包装query实例
          Scorer fragmentScorer = new QueryScorer(query);

          //创建高亮器，高亮效果的实现最终需要通过Highlighter来实现
          Highlighter highlighter = new Highlighter(formatter, fragmentScorer);

          //创建Fragmenter实例，SimpleFragmenter构造器中的传入的参数表示返回的高亮文本片段的最大长度；
          Fragmenter fragmenter = new SimpleFragmenter(10);
          //将格式化片段实例与高亮器关联      
          highlighter.setTextFragmenter(fragmenter);


          /**********高亮相关代码end**********/

          //查询索引表，最终数据都被封装在 TopDocs的实例中
          TopDocs topDocs = indexSearcher.search(query,10);

          //通过topDocs获取匹配全部记录
          ScoreDoc[] scoreDocs = topDocs.scoreDocs;

          for (int i = 0; i < scoreDocs.length; i++) {
              //获取记录的id
              int id = scoreDocs[i].doc;
              //文章的得分
              float score = scoreDocs[i].score;
              System.out.println("id:"+id+" 分章的得分："+score);
              //查询数据表
              Document document = indexSearcher.doc(id);
              //获取标题信息  高亮前title
              String title = document.get("title");
              //获取内容信息   高亮前content
              String content = document.get("content");
              System.out.println(" 高亮前title:"+title+"\t 高亮前content:"+content);

              /*
               *通过高亮器对title和content进行高亮处理  
               *getBestFragment方法中三个参数的含义：1、分词器   2、哪一个字段需要进行高亮   3、需要进行高亮的文本信息 
               *备注：如果需要进行高亮的文本信息不包括查询的关键字则getBestFragment会返回null出去
               * */
              String postTitle = highlighter.getBestFragment(analyzer, "title", title);
              String postContent = highlighter.getBestFragment(analyzer, "content", content);

              System.out.println(" 高亮后title:"+postTitle+"\t 高亮后content:"+postContent);

          }

      } catch (Exception e) {
          // TODO: handle exception
          e.printStackTrace();
      }
  }

}





* 检索结果如下：

高亮前title:旅游景点 高亮前content:长沙岳麓山风景区，风景优美，是一个非常不错的旅游景点。

高亮后title:旅游景点 高亮后content:长沙岳麓山风景区 ```

通过打印的结果发现 旅游 长沙 关键词被<font color='red'></font>标签包裹，意味着将高亮后的信息放在网页中显示即可看到红色字体效果啦。
